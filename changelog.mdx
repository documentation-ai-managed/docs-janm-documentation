## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added advanced search filters for better content discovery
- Implemented real-time collaboration with conflict resolution
- Introduced custom workflow automation for repetitive tasks
- Added support for bulk document operations

## Improvements

- Enhanced API performance with improved caching mechanisms
- Updated user interface for better accessibility and usability
- Improved webhook reliability with retry logic and better error handling
- Optimized database queries for faster search results

<Update label="2024-09-20" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Complete API redesign with RESTful endpoints and improved authentication
- Added project-based organization with granular permissions
- Implemented version control for all documentation changes
- Introduced integration marketplace with popular tools

## Breaking Changes

- API endpoints now require authentication for all requests
- Project structure has been reorganized to support nested folders
- Webhook payload format has been updated for consistency
